# TENG 触觉阵列论文方法与实验章节草稿

> 日期：2026-05-19  
> 目的：把已完成的仿真实验结果整理为论文可直接扩写的正文框架。

## 1. 方法章节草稿

### 1.1 物理启发的 TENG 阵列仿真

本文构建了一个 8 x 8 TENG 触觉阵列仿真框架，用于生成具有时空结构的电压响应信号。每个样本由 64 个 TENG 单元的 200 点时间序列组成，输入张量维度为 8 x 8 x 200。仿真器同时考虑材料相关波形差异、接触力幅值、接触中心位置、接触半径、通道噪声、坏点、增益漂移和串扰等因素，使生成数据能够覆盖触觉阵列在实际采集中的主要变化来源。

接触位置通过二维空间分布映射到阵列单元，接触半径控制激活区域的扩散范围，接触力控制响应幅值并引入饱和趋势。材料类别通过不同的波形形态和幅值参数体现。该设计使仿真数据不仅包含分类任务所需的时间波形差异，也包含定位和接触区域估计所需的空间模式。

### 1.2 数据集构建

实验使用两类核心数据集。v1 数据集作为初始基准数据，包含默认随机分布下的训练、验证和测试样本。v2 augmented 数据集在 v1 基础上扩展了高压力、小半径、大半径、边缘位置和组合困难样本，用于训练默认主模型。

为评估跨分布泛化能力，额外构建了五类固定测试 split：高压力 `force_high_20_30n`、边缘位置 `position_edges`、大半径 `radius_large_12_18mm`、小半径 `radius_small_1_3mm` 和组合困难 `combined_hard`。为评估鲁棒性，构建了 SNR、通道坏点、增益漂移和串扰四类扰动 split。

### 1.3 多任务学习模型

主模型采用空间-时间双分支结构。空间分支从 8 x 8 的峰值、均值、能量和坐标统计图中提取阵列级空间特征，用于定位和接触半径估计。时间分支从代表性通道波形中提取时序特征，用于捕获材料相关动态响应。两个分支的特征拼接后进入共享层，并分别输出材料类别和连续变量回归结果。

模型输出包括材料类别、接触力、接触中心坐标和接触半径。训练目标由材料分类交叉熵和连续变量均方误差组成，并支持 force 维度加权、高压力样本加权和结构变体选择。

### 1.4 训练与模型选择

本文保留五代模型作为实验记录。v1 是默认数据训练的初始模型；v2 使用增强分布训练，是默认主模型；v3 对高压力样本加权，是高压力专项模型；v4 对 force 维度加权，是均衡鲁棒候选；v5 来自短周期网格搜索，仅作为搜索记录。

模型选择不只依据随机测试集，还同时考虑跨分布泛化、鲁棒性和结构消融结果。最终建议以 v2 作为主模型，v3 作为高压力专项对照，v4 作为均衡鲁棒对照。

## 2. 实验章节草稿

### 2.1 仿真合理性验证

首先验证仿真器是否产生符合物理直觉的信号。结果图 `figures/generated/paper/simulation_sanity.png` 展示了材料波形差异、力响应饱和趋势、位置热力图和半径扩散效应。该结果用于说明仿真器能够为后续多任务学习提供具有可解释性的时空信号。

### 2.2 默认随机测试结果

默认随机测试表明，v2 augmented 在连续量估计上优于 v1。v2 的 force MAE 为 2.5548 N、position MAE 为 3.0527 mm、radius MAE 为 0.9711 mm，均优于 v1 的 2.8786 N、3.6620 mm 和 1.1534 mm。材料分类准确率仍处于中等水平，说明材料识别仍是后续优化方向。

### 2.3 跨分布泛化结果

跨分布实验显示，增强分布训练能显著改善困难 split。以 `combined_hard` 为例，v1 的 force MAE 为 9.4394 N，而 v2 降至 6.3855 N；材料准确率也从 0.1750 提升至 0.5250。高压力 split 中，v3 的 force MAE 为 5.6090 N，是当前高压力估计最优模型，但其默认随机测试表现不如 v2，因此更适合作为专项模型。

### 2.4 鲁棒性结果

v2 主模型已经完成 SNR、坏点、漂移和串扰四类鲁棒性评估。结果显示，串扰对当前模型影响较小；坏点比例增加会逐步影响定位与半径估计；强漂移和低 SNR 是主要困难场景。尤其在 `snr_10db` 和 `drift_30pct` 下，radius MAE 分别达到 4.5305 mm 和 4.1706 mm，说明半径估计对强噪声和通道增益变化较敏感。

### 2.5 结构消融结果

结构消融实验验证了双分支设计的必要性。完整模型在 `test_random` 上达到 0.3667 的材料准确率、2.5548 N 的 force MAE、3.0527 mm 的 position MAE 和 0.9711 mm 的 radius MAE。去掉时间分支后，材料准确率降至 0.2500，说明时间波形对材料识别有贡献。去掉空间分支后，position MAE 增至 13.1304 mm，radius MAE 增至 2.4070 mm，说明空间分支是定位和接触区域估计的关键。

### 2.6 局限性

当前系统仍有三点局限。第一，材料识别准确率中等，说明不同材料的仿真波形可分性和时间建模仍可增强。第二，高压力绝对力估计存在饱和导致的误差，需要在传感器物理模型或训练目标中进一步处理。第三，低 SNR 和强漂移下的半径估计明显退化，后续可通过噪声增强、漂移校正或不确定性建模改进。
