# 基于物理启发仿真与多任务学习的 TENG 触觉阵列联合感知研究

> 日期：2026-05-19  
> 状态：论文正文初稿，可继续按目标期刊格式改写。

## 摘要

摩擦电纳米发电机（triboelectric nanogenerator, TENG）触觉阵列能够将接触过程中的机械刺激转化为电信号，在柔性电子皮肤、机器人触觉和自供能传感等方向具有应用潜力。然而，阵列信号通常同时受到接触材料、接触力、接触位置、接触面积以及通道噪声和器件非一致性的影响，单一任务建模难以充分利用 TENG 阵列信号中的时空信息。本文构建了一个物理启发的 8 x 8 TENG 触觉阵列仿真框架，并基于该框架生成多种训练、泛化和鲁棒性测试数据。进一步地，本文提出一种空间-时间双分支多任务网络，同时完成材料识别、接触力估计、接触位置定位和接触半径估计。实验结果表明，增强分布训练得到的 v2 主模型在默认随机测试中取得 2.5548 N 的 force MAE、3.0527 mm 的 position MAE 和 0.9711 mm 的 radius MAE，并在组合困难泛化场景中显著优于初始 v1 模型。结构消融结果进一步表明，时间分支对材料识别有明显贡献，空间分支对定位和接触半径估计至关重要。

## 1. 引言

TENG 触觉传感器能够利用接触起电和静电感应效应产生电压响应，适合构建低功耗、自供能的触觉感知系统。对于阵列化 TENG 传感器而言，单个接触事件会在空间上激活多个单元，并在时间上产生与材料、压力和接触过程相关的动态波形。因此，阵列信号天然包含空间分布与时间响应两类信息。

现有触觉感知研究通常关注单一任务，例如压力估计或接触位置定位，但真实应用中往往需要同时判断接触材料、估计接触力、定位接触中心并推断接触范围。同时，实际阵列还会受到噪声、坏点、通道增益漂移和串扰等因素影响。若只在随机划分数据集上评价模型，难以说明模型在复杂工况下的泛化能力和鲁棒性。

本文围绕 TENG 触觉阵列联合感知建立完整仿真实验流程。主要贡献包括：

1. 构建物理启发的 8 x 8 TENG 阵列仿真器，生成包含材料、力、位置、半径和多类扰动因素的时空信号。
2. 设计空间-时间双分支多任务网络，同时输出材料类别、接触力、接触位置和接触半径。
3. 系统评估默认测试、跨分布泛化、鲁棒性和结构消融，形成可复现的论文实验链路。
4. 明确不同模型版本的用途：v2 作为默认主模型，v3 作为高压力专项模型，v4 作为均衡鲁棒候选，v5 仅作为搜索记录。

## 2. 方法

### 2.1 TENG 阵列仿真框架

本文使用 8 x 8 阵列表示 TENG 触觉传感器，每个阵列单元输出长度为 200 的时间序列，因此单个样本的输入维度为 8 x 8 x 200。仿真器将接触事件分解为材料类别、接触力、接触中心位置和接触半径四个核心因素，并进一步加入噪声、坏点、增益漂移和串扰等扰动。

接触中心决定阵列响应的空间峰值位置，接触半径决定激活区域的扩散范围。接触力影响整体响应幅值，并在高压力区域体现饱和趋势。材料类别通过不同的波形形态和幅值参数体现，使材料识别任务依赖于时间动态特征，而位置和半径估计更多依赖空间分布特征。

仿真合理性图见 `figures/generated/paper/simulation_sanity.png`。该图展示材料波形差异、力响应变化、位置热力图和半径扩散趋势，用于说明生成信号符合基本物理直觉。

### 2.2 数据集与评估 split

实验包含两个主要训练数据版本。v1 是初始默认随机分布数据集，用于建立 baseline。v2 augmented 在 v1 基础上增强高压力、小半径、大半径、边缘位置和组合困难样本，用于训练默认主模型。

除默认随机测试外，本文构建五类跨分布泛化 split：`force_high_20_30n`、`position_edges`、`radius_large_12_18mm`、`radius_small_1_3mm` 和 `combined_hard`。鲁棒性评估覆盖四类扰动：SNR、通道坏点、增益漂移和串扰。该设计使模型不仅在同分布测试上被评价，也在压力范围、位置范围、接触半径和传感器扰动发生变化时被评价。

### 2.3 空间-时间双分支多任务网络

模型结构见 `figures/generated/paper/model_architecture.png`。网络包含空间分支和时间分支。空间分支从阵列信号中提取峰值图、均值图、能量图和坐标图，并通过 2D CNN 得到 96 维空间特征。时间分支选取代表性活跃通道波形，并通过 1D CNN 得到 32 维时间特征。两类特征拼接为 128 维联合表示后进入共享 MLP，并输出材料分类结果和连续变量回归结果。

连续变量包括接触力、接触中心 x/y 坐标和接触半径。训练损失由材料分类交叉熵和标准化连续变量均方误差组成。为了研究模型在高压力和力估计任务上的表现，训练脚本支持高压力样本加权和 force 维度加权。为了验证模型结构贡献，网络还支持 `no_temporal` 和 `no_spatial` 两种结构消融。

## 3. 实验结果

### 3.1 默认随机测试

默认随机测试结果见 `results/paper_tables.md` 和 `results/paper_tables.tex`。相较 v1，v2 augmented 在连续量估计上更稳。v1 的 force MAE、position MAE 和 radius MAE 分别为 2.8786 N、3.6620 mm 和 1.1534 mm；v2 分别降至 2.5548 N、3.0527 mm 和 0.9711 mm。该结果说明增强分布训练没有只改善困难样本，也提升了默认测试中的连续量估计质量。

材料分类方面，v2 的准确率为 0.3667，低于 v1 的 0.4083，但在跨分布困难场景中 v2 的材料准确率更稳定。因此，本文将 v2 作为默认主模型，同时在局限性中说明材料识别仍有提升空间。

### 3.2 跨分布泛化

跨分布泛化结果表明，增强训练和专项加权能够显著改变模型在困难 split 上的表现。在 `combined_hard` 中，v1 的 force MAE 为 9.4394 N，v2 降至 6.3855 N，v3 进一步降至 3.7642 N。该结果表明，高压力样本加权能够有效改善高压力组合困难场景中的力估计。

在 `force_high_20_30n` 中，v3 的 force MAE 为 5.6090 N，是当前高压力场景最优模型。但 v3 在默认随机测试中 force、position 和 radius 表现均不如 v2，因此更适合作为高压力专项模型，而不是默认主模型。v4 在高压力表现和默认测试副作用之间取得折中，可作为均衡鲁棒候选。

### 3.3 鲁棒性分析

v2 主模型的鲁棒性结果覆盖 SNR、坏点、漂移和串扰四类扰动。串扰对当前模型影响较小，说明在当前串扰强度设置下，阵列空间模式仍能被模型稳定识别。坏点比例增加会逐步影响定位和半径估计，但模型没有出现完全失效。

主要短板来自低 SNR 和强增益漂移。在 `snr_10db` 下，v2 的 radius MAE 达到 4.5305 mm；在 `drift_30pct` 下，radius MAE 达到 4.1706 mm。这说明接触半径估计对强噪声和通道增益变化更敏感。若后续继续做增强实验，应优先围绕半径估计的噪声增强、漂移校正或不确定性建模展开，而不是继续扩大普通训练轮数。

### 3.4 结构消融

结构消融结果见 `results/ablation_structural_v1.csv` 和 `figures/generated/ablation/structural_ablation.png`。完整模型在 `test_random` 上取得 0.3667 的材料准确率、2.5548 N 的 force MAE、3.0527 mm 的 position MAE 和 0.9711 mm 的 radius MAE。

去除时间分支后，材料准确率下降至 0.2500，说明时间波形对材料识别具有明确贡献。去除空间分支后，position MAE 增至 13.1304 mm，radius MAE 增至 2.4070 mm，说明空间分支是定位和接触区域估计的关键结构。该结果支持本文采用空间-时间双分支网络，而不是单一空间或单一时间模型。

## 4. 是否还需要继续训练

从论文主线角度看，当前不需要继续训练新模型。已有实验已经覆盖默认测试、跨分布泛化、鲁棒性、结构消融和模型版本对比，足够支撑完整论文初稿。

继续训练只在以下目标下有必要：

1. 若论文目标强调低 SNR 或强漂移工况，需要针对 `snr_10db` 和 `drift_30pct` 下的 radius MAE 做专项增强训练。
2. 若论文目标强调材料识别，需要重新设计材料波形可分性或增强时间分支。
3. 若需要更高最终指标，可进行更大规模网格搜索，但这属于性能优化，不是当前主线实验缺口。

因此，当前优先级应从训练切换到论文正文、图表排版和结果解释。

## 5. 局限性

当前研究仍存在三个主要局限。第一，材料识别准确率中等，说明不同材料的仿真波形差异和时间建模还不够充分。第二，高压力力估计仍受饱和趋势影响，需要更精细的传感器物理模型或损失设计。第三，低 SNR 和强漂移下半径估计退化明显，说明阵列幅值分布对通道扰动敏感。
