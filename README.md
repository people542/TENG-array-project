# TENG 触觉阵列论文项目

本仓库用于管理 TENG 触觉阵列论文项目的规划文档、实验设计、后续仿真代码、模型代码和实验结果。

## 项目方向

基于物理启发仿真与故障鲁棒多任务学习，研究 TENG 触觉阵列的联合感知任务，包括：

- 材料识别
- 压力估计
- 接触位置定位
- 接触半径估计
- 噪声、坏点、漂移和串扰条件下的鲁棒性分析
- 跨分布泛化和不确定性分析

## 当前文档

- `TENG_触觉阵列论文项目完整规划.md`：项目总体规划、研究路线、论文定位和推进计划。
- `TENG_触觉阵列各实验细节完整介绍.md`：各实验的目的、变量、流程、指标、预期结果和论文写法。
- `docs/TENG_仿真参数文献调研.md`：仿真材料参数、阵列参数、噪声扰动范围和参考论文记录。
- `项目进度与交接说明.md`：每次继续项目时优先阅读的上下文交接文件。

## 代码结构

- `simulation/`：TENG 单元、阵列、噪声和数据集仿真代码。
- `scripts/`：运行数据生成、可视化和实验的脚本。
- `tests/`：基础测试和回归检查。
- `docs/`：文献调研、参数依据和论文写作材料。
- `requirements.txt`：当前最小依赖。

## 当前代码进展
